# PELE URBANA
Bem-vindo ao repositório oficial do projeto Pele Urbana, um website de e-commerce dedicado à venda de peças de couro artesanal e autênticas.

#  Visão Geral do Projeto 
 Pele Urbana é mais do que um site de vendas; é uma vitrine digital que celebra a durabilidade e o charme atemporal do couro, unindo o **rústico** da matéria-prima com um toque de **elegância** urbana e moderna. O website foi concebido com uma paleta de cores em tons terrosos, a sofisticação discreta e a origem **orgânica** do material, proporcionando uma experiência visual e focada no produto.

# Funcionalidades Básicas
Este é um projeto inicial focado em demonstrar a estrutura e o estilo visual. As funcionalidades básicas implementadas incluem:

* **Página Inicial (Home):** Apresentação da marca e destaque dos principais produtos/coleções.
 
* **Catálogo de Produtos:** Visualização das peças de couro com imagens e descrições.

* **Design Responsivo (Básico):** Adaptação da estrutura em diferentes tamanhos de tela.

* **Navegação Simples:** Links claros para as diferentes seções do site.

# Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias web fundamentais:

* **HTML5:** Para a estruturação semântica e acessível do conteúdo.

* **CSS3:** Para a estilização completa, aplicação da paleta de cores terrosas e layout.

*  **JavaScript:** Para aplicação de botões.
